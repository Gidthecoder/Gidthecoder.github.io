Introduction to Javascript

Prequisite: Knowledge in basic computer literacy

back ←next →

Javascript is a programming language that runs on web browsers.

It's used on the client-side on a webpage for adding interactivity and accessing various web APIs(database, networking, caching, audio, location,...)

With Nodejs, Javascript can be used for Backend development, App development, Game Development, machine learning, AI, Computer graphics,...

Installing and running JavaScript

Javascript runs on a web browser so there's no need to install any software except your web browser eg Google chrome.

You can use the console tab on the developers tools to test some codes.

Javascript can be linked to HTML internally using the script tag

An external Javascript file (with the .js extension) can also be linked to HTML using the script tag with src attribute

Javascript syntax

Variables are used for storing values.

Javascript can store different type of data

Comments are texts that won't be executed.

function is a reusable block of code that performs a single task

Javascript has various Built-in functionalities for manipulating data
