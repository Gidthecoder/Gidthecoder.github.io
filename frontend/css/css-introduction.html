Introduction to CSS

Prequisite: Knowledge in basic computer literacy, Basic knowledge in HTML

back ←next →

CSS means Cascading Style Sheets.

It's a stylesheet language for styling markup languages like HTML, XML and SVG

CSS can add colors, change layouts, change fonts,... of HTML elements.

CSS can also make your web page look responsive on various devices

CSS files

CSS files can be created and modified with a text editor like Notepad, VS code.

CSS files must be saved with the .css extension.

Linking CSS to an HTML document

CSS files can be linked externally by using the link element

CSS can be linked internally by using the style element

CSS can be used inline by using the style attribute

Syntax and structure of CSS

CSS documents are made up of rules

CSS rules are made up of selectors and declarations

Selectors describes the elements to style

declarations are made up of properties

Properties describes the specific style to apply on the selectors

CSS selectors

Selectors describes the elements to style

You can apply styling on a single type of element

You can apply styling on a list of elements

You also can target specific elements

CSS properties

Properties describes the specific style to apply on the selectors

it starts with a property name, followed by a colon(:), followed by a property value and ends with a semicolon(;)

There is a long list of CSS properties eg color, font-size, background-color, border, padding,...

CSS comments

Comments are text content that won't be executed.

It's used for explaining logic behind some CSS styling

It's syntax is /* comment */
