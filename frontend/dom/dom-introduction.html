Introduction to DOM

Prequisite: Basic computer literacy, Knowledge in HTML, CSS and JavaScript

back ←next →

DOM means Document Object Model.

It represents a web page in form of an hierachial tree so that it's structure and content can be modified by scripting languages like javascript

DOM represents the document as object and stores the methods for manipulating it in an interface.

JavaScript is the default programming language for manipulating the DOM

Data types

Every object in a document is associated with a (list of) data type.

The Document type refers to the root document

The Element type refers to elements object within the documents

The Node type refers to all objects(Elements, Comments, Attributes) within the document

The data types is associated with a (group of) interfaces for manipulating it

There are many other type of data types and interfaces

DOM interfaces

DOM Has several interfaces for manipulating different data types

The Document interface has methods for manipulating the Document type

The Node interface has methods for manipulating Node type

The Element interface has general methods for manipulating the Element type

There are also multiple specialized Interfaces for manipulating HTML elements(which is a subset of Element type)

Multiple interfaces can belong to a single data type

A method for an interface can either return an object, data type or perform a task on the document

Inheritance

Various interfaces inherits methods and properties from some top-level interfaces

Popular DOM methods

Document Interface: querySelector(selector) method selects the first element that matches the selector.

Element interface: textContent(html) method add or modifies the text content of an Element.

HTMLElement interface: style property modifies the CSS styling of HTML elements
