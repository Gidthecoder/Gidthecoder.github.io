Introduction to HTML

Prequisite: Basic computer literacy

back ←next →

HTML means Hypertext Markup Language.

It's a markup language for defining the structure and content of a webpage.

HTML can be used to add texts, links, images, videos, tables,... to a webpage.

Every website on the internet is built with HTML.

HTML files

HTML files can be created and modified with a text editor like Notepad, VS code,...

HTML files must be saved with the .html extension.

HTML files can be viewed from a web browser.

Structure of an HTML document

HTML documents are made up of elements.

Elements describes the type of content to store.

HTML elements are made up of tags and content.

Tags defines the start and end of the element.

Attributes can be used to modify the behaviour of the element.

Every HTML documents must start with this format:

<Doctype html> - declaration tells web browsers that the HTML page follows HTML5 syntax.

html element - is the root element for the document.

head element - stores document metadata that'll be used by other parties eg search engine, social media,...

body element - stores the actual content that'll be visible to the user.

HTML elements

Elements describes the type of content to store.

HTML elements are made up of tags and content.

HTML elements can be open or closed element

Open HTML elements must start with a start tag followed by the content and end tag.

Closed elements only has a start tag

HTML tags

Tags defines the start and end of an element

Tags are divided into start tag and end tag

Start tag starts with < followed by the tagname(eg p) and ends with a >

End tag starts with < followed by the tagname(eg p) and ends with a slash / and >

Tags are case-insensitive.

HTML attributes

HTML element modifies the behaviour of an element

If present, must be stored within the start tag (after the tag name)

Attributes starts with an attribute name followed by an equal sign = and attribute value

Attributes can be global or content attribute

Global attributes can be applied to all elements

Content attribute can be applied to specific elements

HTML comments

Comments are lines of text that aren't executed by the web browser.

comments starts with <!-- followed by the content and ends with -->
