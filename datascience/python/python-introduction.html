Python is a programming language created by Guido van rossum in 1991.

It can be used for backend development, Data science, AI, software development and many more.

Installation and editing

Download a version of python (at https://python.org) that matches your system specification and install.

python files (saved with '.py' extension) can be created and edited with an IDE( like IDLE python) or text editor (like Notepad, VS code)

Running python

Use your system's command line interface to navigate to your python file and enter py Name-of-file.py or pythom Name-of-file.py

Or run it from an IDE like IDLE python

Interactive python

To test some python features quickly, open command line and enter 'python' or 'py'.

Or open IDLE Shell

Python syntax

Variables are used for storing values.

Python can store different type of data

Comments are texts that won't be executed.

function is a reusable block of code that performs a single task

Python has various Built-in functionalities for manipulating data

Python Package Manager

A Package manager is used creating and managing libraries.

A library extends the functionalities of Nodejs. Libraries can be created for Web servers, accessing database, API integration, testing codes, creating, building games,...

Python uses pip(by default) as package manager

Some popular libaries for python includes Django, Numpy, Tensorflow, Beautifulsoup
