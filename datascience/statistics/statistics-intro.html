Introduction to Statistics

Prequisite: Analytical skills, Critical thinking, ... and other important soft skills

back ←next →

Statistics is the collecting, organizing and analyzing of data in order to get useful information

Statistics is used in every area from management to science to research,...

Knowledge of Statistics is important for every data scientist

Statistics can be used to analyze the following: sales, sport data, government data, scientific phenomenon, household finance, global phenomenon,...

Data

Data is a piece of information.

Data can be Numerical or categorical

Numerical data are quantitative data eg age, score, height, sales

Categorical data are qualitative data eg Male/female, yes/no, soda/alcohol

Numerical data

Numerical data is divided into discrete and continous data

Discrete data are items that can be counted eg 1, 2

Continous data are items that can't be counted. They are mostly measurement figures eg gallons

Data set

A data set is a collection of data

Variable and Distributions

A variable is a characteristics that is being counted, measured or analyzed. eg age, height, sales,...

A distribution is a listing of the possible values of a variable

Population

Population is a collection of people or object of interest eg Nigerian citizens.

Sample

A sample is a subset of the population. it's used when it's costly or impossible to get every data on the population

Statistics

Statistics are summaries of the data collected from a sample.

Parameters

Parameters are summaries of the data collected from a population.

Levels of data

Nominal level - are used for classification.

Ordinal level - can also be used for ranking and ordering of objects. do you like rice?

Interval level - are always numerical, it can be negative value and it doesn't have absolute zero. eg Celcius

Ratio level - are always numerical, it can only be positive value and it can have absolute zero. eg height, Age

Steps in statistical Analysis

Pick a research question

Collect quality data

Summarize and analyze data

Use summaries to draw a conclusion about the original question.

Descriptive statistics

This is when you use data gathered on a group to reach a conclusion for that same group.

Inferential statistics

This is when stats generated on a sample is used to make a conclusion on the population

Charts

Charts helps summarize data in a visually appealing way.

Margin of error

Margin of error measures how many % point your sample result will differ from the population value

Confidence Interval

Confidence interval is the % of time you are confident that your sample result will fall within a range of values when you resample the population

Hypothesis Testing

This is a technique for using data to validate or invalidate a claim about a population

Correlation and Regression

Correlation measures the strength of relationship between 2 variables

Regression uses one (or more) variables to make a prediction on another variable.
