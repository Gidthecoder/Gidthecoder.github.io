Nodejs

Introduction to Nodejs

Prequisite:Basic computer literacy, Knowledge in JavaScript

back ←next →

Nodejs is runtime environment that helps run JavaScript outside the browser

It allows you to use js for backend development, Data science, AI, software development and many more.

Installation and editing

Download a version of Nodejs (at https://nodejs.org) that matches your system specification

Text editors like Notepad, VS code,... can be used to create and edit JavaScript files (.js)

Running Nodejs

Use your command line interface to navigate to your js file and enter node Name-of-file.js

Nodejs syntax

Since Nodejs is a runtime environment for js, it uses the same format

Nodejs uses and has modules for importing various built-in and custom functionalities.

Nodejs Package Manager

A Package manager is used for creating and managing libraries.

A library extends the functionalities of Nodejs. Libraries can be created for Web servers, accessing database, API integration, testing codes, creating, building games,...

Nodejs uses npm(by default) or yarn as package manager

Some popular libaries for JS includes Expressjs, d3.js, React, axios
